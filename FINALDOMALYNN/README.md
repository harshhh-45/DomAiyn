# DomAIyn Labs - Space Themed SPA

A production-grade, enterprise-level landing page for DomAiyn Labs. Built with a futuristic space theme, featuring smooth animations, 3D visualizations, and a responsive design.

## 🚀 Features

- **Futuristic UI**: Modern space-themed aesthetics with glassmorphism and smooth gradients.
- **3D Visualizations**: Highly interactive 3D particle systems and molecular structures built using `@react-three/fiber` and `@react-three/drei`.
- **Smooth Animations**: Sophisticated scroll-triggered animations using `framer-motion`.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **Security Focused**: Content Security Policy (CSP) headers and privacy-first architectural descriptions.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **3D Graphics**: [Three.js](https://threejs.org/) / [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd domaiyn-labs
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components`: UI components (Hero, About, Bayora, Contact, etc.)
- `src/shaders`: Custom GLSL shaders for visual effects.
- `public/`: Static assets (logos, icons).
- `src/App.jsx`: Main application layout.
- `src/index.css`: Global styles and Tailwind configuration.
